# personal_prompts

Prompts for working with AI coding assistants as a learning partner, not just a code generator.

## Philosophy

Use AI to code *and* learn. Each prompt is designed to be concise and practical, always extracting reasoning, trade-offs, and targeted insights—not just solutions.

## Structure

- Each prompt topic is kept in a dedicated `.prompt.md` file for clarity and ease of use.
- All prompts are designed to:
  - Be concise and focused (never overengineered)
  - Encourage thoughtful decisions and highlight risks
  - Be practical for real-world software teams
