# personal_prompts

Prompts for working with AI coding assistants as a learning partner, not just a code generator.

## Philosophy

Use AI to code *and* learn. Every prompt is designed to extract reasoning, trade-offs, and real-world lessons—not just solutions.

## Structure

- `senior-engineer-prompts.md` — Concise, token-efficient prompts covering:
  - Implementation, Architecture, Code Review, Spikes/POCs
  - Documentation, Refactoring, Debugging, Incidents
  - Performance, Security, Testing, API Design, Tech Leadership
  - And more...

Each prompt asks AI to explain *why*, not just *what*.
